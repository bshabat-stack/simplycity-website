Files in this folder whose names start with an underscore are ignored by the
blog, so this template never shows up on the site.

To publish a post: copy this file, name it after the URL you want
(`why-reviews-matter.md` becomes `/blog/why-reviews-matter` — lowercase letters,
numbers and hyphens only), fill in the frontmatter above, and write the article
below it in Markdown. That's the whole process — no code changes needed.
